# Negative — AWS lookalikes

`AKIA` prefix needs 16 uppercase-alphanumeric chars. These are too short or too long or contain lowercase.

- `AKIAshort` (too short)
- `AKIAIOSFODNN7EXAMPLELONG` (too long — 20)
- `AKIAiosfodnn7example` (lowercase in body)

Also: words containing "AKIA" inside prose should not match.

> The akiaAWS documentation shows AKIAABCDEFG (only 7 chars) as a truncated example.
